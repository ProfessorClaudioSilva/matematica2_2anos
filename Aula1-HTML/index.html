Introdução ao HTML

HTML (HyperText Markup Language), ou Linguagem de Marcação de Hipertexto, é a base para criar páginas na internet. Ele organiza textos, imagens, links, vídeos e outros elementos em uma página web.

Como funciona?

O HTML usa tags para marcar cada parte do conteúdo. As principais tags são:

  • <html>: Indica o início da página HTML.
  • <head>: Contém informações sobre a página, como o título.
  • <body>: Onde fica o conteúdo visível da página.
  • <h1> até <h6>: Títulos e subtítulos.
  • <p>: Parágrafos de texto.
  • <a>: Links para outras páginas.
  • <img>: Imagens.

Exemplo básico:

<!DOCTYPE html>
<html>
  <head>
    <title>Minha primeira página</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
    <p>Este é um exemplo de página HTML.</p>
  </body>
</html>

Dicas rápidas:

  • As tags geralmente têm uma abertura (<tag>) e um fechamento (</tag>).
  • O conteúdo da página vai dentro do <body>.
  • Você pode abrir arquivos .html no navegador para ver o resultado.
